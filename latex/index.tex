\hypertarget{index_intro}{}\doxysection{Úvod}\label{index_intro}
Program slouží k převodu z desítkové soustavy do Osmičkové a hexadecimální.\hypertarget{index_motiv}{}\doxysection{Jak program funguje?}\label{index_motiv}
Z uživatelského hlediska\+:
\begin{DoxyItemize}
\item spustíme E\+XE soubor
\item zadáme vstup v desítkové soustavě
\item vybere soustavu, do které budeme vstup převádět
\end{DoxyItemize}

Z programétorského hlediska
\begin{DoxyItemize}
\item Jsou definované 4 funkce
\item pomocí knihovny Colorful console se vykreslí logo programu
\item funkce startconv provádí interakci s uživatelem
\item po zadání vstupu se dle výběru spustí vybraná funkce se vstupním parametrem
\item vybraná funkce provede výpočet
\item vybraná funkce zavolá num\+Conv(), která převede číselné typy do typu string
\item vybraná funkce vypíše výstup 
\end{DoxyItemize}
